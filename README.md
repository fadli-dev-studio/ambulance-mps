# AMBULANCE MUHAMMAD PUTRA SERVICE

Blueprint awal landing page statis berbasis **HTML/CSS/JS sederhana** untuk bisnis jasa ambulans **AMBULANCE MUHAMMAD PUTRA SERVICE**.

Status saat ini:
- riset konten sudah disiapkan lewat **NotebookLM MCP**
- arah desain visual sudah digenerate di **Stitch MCP**
- implementasi website belum dimulai

## Tujuan Project

Membuat landing page yang fokus pada:
- memperkenalkan layanan ambulance yang tersedia
- membangun rasa percaya calon pelanggan
- memudahkan pengunjung melakukan kontak cepat via telepon / WhatsApp
- menjadi fondasi untuk website company profile sederhana berbasis HTML statis

## Referensi Utama
